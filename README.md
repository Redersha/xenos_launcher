# 什么神人也配写README
基于 node 的 Minecraft:Java Edition 启动器。TUI界面，拒绝花里胡哨

*算是个 vibe coding。AI的贡献量得有70%了。*

## 我要玩
点右侧 Release 区域下的最新版本，或者 https://github.com/Redersha/xenos_launcher/releases/tag/v0.3
支持 Windows x64、macOS arm、macOS x64、Linux

## 功能
- 从modrinth curseforge下载资源
- 下载游戏，从1.0到最新的快照版都有
- 自动下载合适的JDK - 用azul的
- JDK管理
- 收藏你喜欢的模组
- 支持正版账户、LittleSkin、离线账户
- 自动内存分配 - 同时开一万个游戏也没事，只要你的RAM够大……
- 自动更新启动器版本
- 不太好用的命令行系统 - 敲键盘绝对比界面快！
- 支持Forge、NeoForge、Quilt、Fabric的实例创建
- 将模组的描述翻译成中文 - 有点机翻罢
- 超级快的启动！
